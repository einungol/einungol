### [Blue Team Platform](https://github.com/einungol/blue-team-platform) &nbsp;·&nbsp; [🌐 Live Demo](https://blue-team-platform-ecru.vercel.app)

A self-built, TryHackMe-style **cyber security training platform** focused on defensive skills. Full-stack, deployed, and playable in the browser.

- **10 guided rooms + 7 hands-on terminal labs** covering log analysis, phishing triage, network/C2 forensics, memory forensics, SIEM/Splunk, malware triage, Active Directory attacks, ransomware IR, AWS/cloud logs, and detection engineering.
- **Simulated Linux terminal** (sandboxed) where users `grep`/`cut`/`sort` through real log artifacts to capture flags — no Docker required.
- **Learning Paths**, achievements, leaderboard, and server-authoritative answer checking.
- **Stack:** Next.js 16 · TypeScript · Node/Express · Tailwind — deployed on Vercel + Render.
